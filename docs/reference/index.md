# Package index

## Main functions

- [`geom_jitter_gauss()`](https://github.com/natydasilva/jitter2D/reference/geom_jitter_gauss.md)
  : Jittered 2D points based on Bivariate normal distribution
- [`geom_jitter_quasi()`](https://github.com/natydasilva/jitter2D/reference/geom_jitter_quasi.md)
  : Jittered 2D points based on Sobolev sequence
- [`geom_jitter_quasiloc()`](https://github.com/natydasilva/jitter2D/reference/geom_jitter_quasiloc.md)
  : Jittered 2D points based on Sobolev local sequence
- [`geom_jitter_quasitog()`](https://github.com/natydasilva/jitter2D/reference/geom_jitter_quasitog.md)
  : Jittered 2D points based on Sobol sequence (toggle local/global)
- [`position_jitter_gauss()`](https://github.com/natydasilva/jitter2D/reference/position_jitter_gauss.md)
  : 2D Jitter points to avoid overplotting based on Gaussian bivariate
  distribution
- [`position_jitter_quasi()`](https://github.com/natydasilva/jitter2D/reference/position_jitter_quasi.md)
  : 2D Jitter points to avoid overplotting based on Sobolev sequence
- [`position_jitter_quasiloc()`](https://github.com/natydasilva/jitter2D/reference/position_jitter_quasiloc.md)
  : 2D Jitter points to avoid overplotting based on Sobolev local
  sequence
- [`position_jitter_quasitog()`](https://github.com/natydasilva/jitter2D/reference/position_jitter_quasitog.md)
  : 2D Jitter points to avoid overplotting based on Sobol sequence
  (toggle local/global)
- [`dayles`](https://github.com/natydasilva/jitter2D/reference/dayles.md)
  : A toy data set for testing methods
